# Serene

**Serene** is a modern StarterKit for full-stack web development using **Spring Boot** and **Angular**. Inspired by Laravel Breeze, it provides a **clean**, **secure**, and **minimalist** foundation so you can focus on building features â€” not spending time on initial configuration.
